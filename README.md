# flappy-bberd

Python3 quick and dirty flappy bird clone



## Keybinds
| Key    | Action         |
| ------ | -------------- |
| Space  | Jump           |
| P      | Pause          |
| Escape | Quits the game |

## Installing dependencies
```bash
pip install -r requirements.txt
```

## Running

```bash
python App.py
```
